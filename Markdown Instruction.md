## Списки

Чтобы добавить ненумерованные списки,необходимо пункты выделить звёздочкой(*)
Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить немерованные списки,необходимо пункты просто пронумеровать.
Напримеер, вот так :
1. Первый пункт
2. Второй пункт
