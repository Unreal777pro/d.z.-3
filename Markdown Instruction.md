# Инструкция для работы в Markdown

## Выделение текста

Чтобы выделить текст курсивом,необходимо обрамить его звёздочкой (**).Например: *вот так*

Чтобы выделить текст полужирным,надо обрамить его двойными звёздочками (**)
Например **вот так**
